Sistema de controle de versão

instalar 
yum install git
criar pasta de repositorio local
entrar na pasta
git config --global user.email "you@example.com"
git config --global user.name "Your Name"
dar 
git init
criar arquivos
git add .
git commit -m "teste"

obs:por padrão monitora as modificações
.gitignore para limitar os arquivos que devem ser ignorados

quando modificar
git add arq
git commit -m "com"

git log para verificar os commits

git log -p -2 dois ultimos commits

git diff arq verifica diferenças de modificações

git checkout teste4 volta as modificações antes do commit

git rm teste4

git tag -a V1.1 -m "Versao 1.1"
git tag

git log --pretty=oneline

branch , ponteiro , area de trabalho

git branch novo cria novo branch

muda para outro branch
git checkout novo
